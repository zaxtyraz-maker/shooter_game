# Shooter Game
## Deskripsi singkat
Ini adalah projek game tema arkade yang dibuat menggunakan Python 3.11.6 menggunakan library Pygame

## Tujuan projek
Projek ini dibuat untuk bersenang senang dan berkreasi menggunakan Python

## Menjalankan projek
Untuk menjalankan game, kalian harus download file .exe dan semua assetnya. Code ada di file shooter_game.py silahkan jika ingin modifikasi.
